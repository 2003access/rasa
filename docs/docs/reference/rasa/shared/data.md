#### is\_likely\_yaml\_file

```python
is_likely_yaml_file(file_path: Text) -> bool
```

Check if a file likely contains yaml.

**Arguments**:

- `file_path` - path to the file
  

**Returns**:

  `True` if the file likely contains data in yaml format, `False` otherwise.
