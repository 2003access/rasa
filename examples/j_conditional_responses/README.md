# Conditional Response Bot DEMO

Typical conversation:
```
Your input -> hello
Hello!
Your input -> what do you do?
I can tell you the time.
Your input -> what is it?
It is Wednesday, 30. June 2021 12:07PM
Your input -> what else can you do?
As I've said, I can tell you the time.
Your input -> anything else?
I really can only tell you the time. Nothing more!
Your input -> Sure. What else can you do?
It's too late to tell you.
Your input -> bye
Bye! Enjoy the rest of your day!
```
